Installation
Use LINUX Command-line, and navigate to the folder 'Project 15'.
Install the packages: Node, and Express.

Usage
1. Once the packages are installed, enter the command: node main.js
2. The Command-Line will return the IP Address and Port number where the Server is hosting the Website.
3. Open your browser and type the given IP Address:Port. Example: 192.168.1.7:8000
4. Explore the Application and conduct custom/random search queries!
